# Text Field

## Installation

```jsx
import { TextField, SignalingLevel, TextAlign, CommitType } from 'oe-library';  
```

## When to Use

The Textfield is a UI Component for displaying and editing textual information. In addition to alphanumeric characters, the input must also allow the input of powers and fractions, etc.

The Textfield is available in two different versions:

* Read-Only Textfield
* Editable Textfield

A Textfield can be combined with a Label. It can be surrounded with an Alarm
Border or a Focus Border. If the control is focused, the Focus Border is added
to the dimension of the control.
