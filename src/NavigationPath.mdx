# Navigation Path

## Installation

```jsx
import { NavigationPath } from 'oe-library';  
```

## When to Use

The Navigation Path is used to demonstrate the depth of the hierarchy and/or to
ensure a more precise naming of the displayed information.

The Navigation Path is always available in an application with hierarchy levels.
The root element of the Navigation Path uses an icon to indicate whether the
user is in the menu area or in the topological area (house or tile icon). For
the use of a single machine, the root element is named with the machine name.
For larger applications with upstream levels (e.g. hall or factory), the name of
the relevant element is used (e.g. 'Hall 1' or 'Shanghai').
