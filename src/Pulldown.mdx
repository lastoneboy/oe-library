# Pulldown

## Installation

```jsx
import { PulldownMenuButton, PulldownGroup, PulldownItem } from 'oe-library';  
```

## When to Use

A Pulldown is used to list a number of choices. It is composed of the background
and several Entries. Entries might be grouped by a Groupname. Selecting an Entry
initiates the command to make the selection.

The Entries can be used with or without grouping in the Pulldown.
