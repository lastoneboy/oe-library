# Oerlikon Component Library

## Installation

Install the library as a npm dependency using `npm` or `yarn`.

```
npm install --save oe-library
```

## Usage

Import any React component the module exports to use it. Required CSS will be injected automatically.

```
import { Switch } from 'oe-library';

// ...

<Switch value={true} onChange={value => console.log(value)} />
```
