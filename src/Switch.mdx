# Switch

## Installation

```jsx
import { Switch } from 'oe-library';
```

## When to Use

A Switch is a UI Component for displaying an on/off state.

A Switch is always displayed with a label or icon. The Label takes over the
function of the exact description for the Switch and can therefore become very
long. To ensure good legibility, grouped Switches should stand vertically one
below the other.

## Examples
