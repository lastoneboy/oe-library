# Radio Button

## Installation

```jsx
import { RadioButton } from 'oe-library';
```

## When to Use

The Radio Button is a UI Component used to make an exclusive selection of
multiple options. When the new selection is accepted, the previous selection is
removed.

A Radio Button cannot stand alone and must be combined with other UI Components,
such as a Label or an icon.

In a group of Radio Buttons at least two options must be offered. The first
Radio Button should have the default selection if possible.

The Label or the icon take over the function of the exact description for the
Radio Button. The Label can therefore be very long. To ensure good legibility,
grouped Radio Buttons should be placed vertically one below the other.
