# Command Button

## Installation

```jsx
import { CommandButton, SignalingLevel } from 'oe-library';
```

## When to Use

A Command Button is used to trigger an action. A Command Button can be combined
with other elements, e.g. an alarm state and/or with the Focus Border. The alarm
border and Focus Border are used in addition to the existing state of the button
(alarm border inside dimensions, focus border outside dimensions).
