# Table

## Installation

```jsx
import { Table } from 'oe-library';
```

## When to Use

A Table is a UI component that arranges data or objects in a clearly structured
representation of horizontal rows and vertical columns.

<Note>
  The <code>Table</code> component provides functionality for resizing columns.
  However, <strong>it does not resize the columns itself.</strong> New column
  widths are passed to the <code>onResizeColumn</code> callback and need to
  be applied to the <code>columns</code> configuration by the component user.
</Note>
