# Combo Box

## Installation

```jsx
import { ComboBox } from 'oe-library';
```

## When to Use

The Combo Box is a UI Component that offers the user a list of entries or options.

The list of entries or options is visualized by a Dropdown.

## Examples

import { ComboBox, _ComboBox } from './ComboBox'

<Playground>
